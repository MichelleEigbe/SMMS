# SMMS
A PLC-based smart motor monitoring system that collects temperature and speed data, streams it through an industrial data pipeline, and visualizes system performance through a real-time dashboard with predictive maintenance capabilities.
# Mission Objective
Build a smart motor monitoring system that can:

* Collect motor temperature and speed data using a PLC
* Stream operational data through an industrial integration pipeline
* Store system data for monitoring and analysis
* Display real-time performance metrics on a dashboard
* Detect abnormal motor behavior using anomaly detection
